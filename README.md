# OpenWrt Builder for Redmi AX6000

Автоматическая сборка OpenWrt 25.12.5 (ImageBuilder) для Xiaomi Redmi Router AX6000.

## Возможности
- Stock + Ubootmod layout
- uci-defaults (WiFi пароль 1234567890, IP 192.168.2.1, русский язык)
- Быстрая сборка через ImageBuilder
